Problem Statement

Tracking personal health metrics such as weight, calorie intake, and exercise routines can be tedious without a simple and organized system. Users need an easy-to-use tool that allows them to record and review these important health data points consistently over time.

Scope of the Project

This project focuses on developing a lightweight, command-line based health tracking application that enables users to log daily weight, calorie consumption, and exercise activity. It stores data in separate files and provides options to view the recorded information whenever desired. The scope is limited to basic logging and retrieval functionalities without advanced analytics or integration with other platforms.

Target Users

Individuals interested in monitoring their personal health metrics regularly

Beginners looking for a straightforward tool to maintain exercise and nutrition records

Users comfortable with using simple command-line applications

High-Level Features

Record daily weight with date stamps
Log daily calorie intake
Capture exercise type and duration
View recorded data for weight, calories, or exercise
